## 🌌 About the Project

**The Vanguard** is a dual-threat project. It features a high-performance automation CLI designed to streamline workflows for modern developers, backed by a highly stylized, dark-tech web portfolio. We merged raw utility with Jujutsu Kaisen and Dragon Ball Z aesthetics to prove that developer tools don't have to be boring.
